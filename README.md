# D200_PS2
Discrete Choice Modelling in Python using the Random Utility Model (RUM). choice-learn multinomial logit and RUMnet models.


## Installation

```zsh
python -m venv venv_d200_ps2_py
source venv_d200_ps2_py/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
